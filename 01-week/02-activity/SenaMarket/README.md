#  SenaMarket
## Instructror : Jesús Ariel González Bonilla 
## Aprendiz : Merari Urbano Quintero

## Descripción

Este proyecto permite visualizar una tienda virtual sencilla donde se pueden visualizar productos, realizar compras y gestionar información básica, el objetivo es que el usuario pueda visualizar la información de la tienda y realizar compras, todo esta muy basico y sencillo, pero que permita visualizar el minimo funcionamiento de la aplicación.
